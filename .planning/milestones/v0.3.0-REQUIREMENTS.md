- [x] **SAFE-01**: `conjure init --dry-run` performs zero filesystem mutations — dry-run threads through `init-project.sh`, profile `apply.sh`, and the `.conjure-version` stamp (fixed live bug)
  - *Outcome: Validated — Phase 2, lib/mutate.sh chokepoint*
- [x] **SAFE-02**: All writes route through one shared mutation helper (`lib/mutate.sh`) that honors dry-run
  - *Outcome: Validated — Phase 2, all 9 profiles + 4 overlays + init retrofitted*
- [x] **SAFE-03**: Generated hook wiring runs on native Windows — init emits portable `node .mjs` hook wiring (fixed live bug in `templates/settings.json.tmpl`)
  - *Outcome: Validated — Phase 1, bash hooks replaced by node/.mjs universally*
- [x] **SAFE-04**: Pre-flight check reports each missing dependency with a copy-pasteable, OS-detected install fix-it and never auto-installs
  - *Outcome: Validated — Phase 1, scripts/preflight.sh extracted with brew/apt/winget/npm hints*

### Cost Estimation

- [x] **COST-02**: Cost output is labeled an estimate with explicit ±band and names model + pricing date
  - *Outcome: Validated — Phase 6, no false precision; labeled "ESTIMATE"*

### Telemetry

- [x] **TLMY-01**: Skill-firing telemetry is opt-in (off by default) and PII-free

- [x] **TLMY-02**: Telemetry writes local-only append-only JSONL the user owns (`.claude/telemetry/`) with zero network egress

- [x] **TLMY-03**: A build/CI test greps all shipped hooks to assert no network egress from telemetry

- [x] **TLMY-04**: Conjure produces a skill "retire-list" from the local telemetry event log
