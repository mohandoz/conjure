<objective>
Create scripts/publish-skill.sh — the worker script for conjure publish-skill.

This is a new POSIX bash worker modelled exactly on scripts/publish-plugin.sh. It runs four sequential gates:
1. Prerequisites (git installed, SKILL.md exists) — exit 2 on failure
2. SHA-pinning guards (skill committed, conjure tagged) — exit 1 per D-07/D-08
3. Frontmatter + size validation against skill.schema.json constraints — exit 1
4. Static egress scan of the skill body — exit 1 per D-01/D-02

After all gates clear, it prints either a gh pr create command (when gh is present) or a manual URL + checklist (when gh is absent). --to flag substitutes the target repo per D-05. Calls mutate_summary at the end for DRY_RUN=1 accounting.

Purpose: Deliver SKILL-01 (validation), SKILL-02 (PR instruction printing), SKILL-03 (SHA-pinning), SKILL-04 (--to flag).
Output: scripts/publish-skill.sh, executable, shellcheck-clean.
</objective>
