## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI → audit-setup.sh | User-supplied target path and flags enter bash subprocess via env vars |
| audit-setup.sh → lib/exact-count.mjs | CONJURE_HOME-derived path invokes Node subprocess; ANTHROPIC_API_KEY read from env |
| lib/exact-count.mjs → Anthropic API | Network call with API key credential (opt-in path only) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-01 | Information Disclosure | lib/exact-count.mjs | mitigate | ANTHROPIC_API_KEY read from environment variable, never passed as CLI arg or written to stdout; key is not echoed in any advisory message |
| T-06-02 | Tampering | scripts/audit-setup.sh (path to prices.json) | mitigate | CONJURE_HOME is self-derived from $0 (absolute path set by cli/conjure); find is scoped to .claude/ within the audited target, not user-supplied paths |
| T-06-03 | Tampering | lib/exact-count.mjs (target path) | mitigate | target is process.argv[2] from cli/conjure which already cd-validates it; find is scoped to target/.claude/ |
| T-06-04 | Elevation of Privilege | cli/conjure cmd_audit (flag injection) | accept | --cost and --exact map to integer env vars (0 or 1) only; no shell expansion in flag values; no exec of user-supplied commands |
| T-06-05 | Denial of Service | lib/exact-count.mjs (large .claude/ tree) | accept | Token count API is rate-limited by Anthropic; no unbounded local computation |
| T-06-SC | Tampering | npm install @anthropic-ai/sdk | mitigate | Package legitimacy audit completed in RESEARCH.md (19.3M/wk, anthropics org, no postinstall); package is not bundled — user installs on demand only for --exact path |
