       - Check `if [ "${CONJURE_EXACT:-0}" = "1" ]; then`

       - fi

    B. Total cost via awk (%.2f for display):
       `TOTAL_COST=$(awk "BEGIN {printf \"%.2f\", $TOKENS_TO_USE * $PRICE_INPUT / 1000000}")`

       - `trap 'rm -f "$COST_TMP"' EXIT`

         - `tokens=$((chars / 4))`
         - `cost=$(awk "BEGIN {printf \"%.6f\", $tokens * $PRICE_INPUT / 1000000}")`

       - For each SKILL.md: `while IFS= read -r skill; do ... done < <(find .claude/skills -name SKILL.md 2>/dev/null)`

         - Read sorted rows: `sort -t' ' -k4 -rn "$COST_TMP" | while IFS=' ' read -r name chars tokens cost; do printf "  %-30s %8s %8s  \$%10.6f\n" "$name" "$chars" "$tokens" "$cost"; done`
       - Print TOTAL footer: `printf "  %-30s %8s %8s  \$%10.2f\n" "TOTAL" "$TOTAL_CHARS" "$TOKENS_TO_USE" "$TOTAL_COST"`

       `echo "  Estimate: \$$TOTAL_COST ±${BAND_PCT}% (chars/4 heuristic · prices: $PRICING_DATE · model: $MODEL)"`
