# Phase 15.1 Summary: Fix release.yml Docker+Homebrew coupling

**Completed:** 2026-05-26
**Plans:** 1 (15.1-01)

## What Changed

**`.github/workflows/release.yml`** — restructured from 2-job to 4-job pipeline:

| Before | After |
|--------|-------|
| `ci-gate` → `release` (7 sequential steps) | `ci-gate` → `release` → `docker` + `homebrew` (parallel) |
| Docker failure silently skips Homebrew | Each job fails independently |
| No token preflight | Homebrew preflight warns + skips on missing secret |
| `packages: write` on release job | `packages: write` only on docker job |
